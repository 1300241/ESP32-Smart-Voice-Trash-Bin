# 硬件接线说明

## 开发板

ESP32-WROOM-32 开发板

## 引脚分配

### 按键 (INPUT_PULLUP, 按下为低电平)

| 功能 | ESP32 GPIO |
|------|-----------|
| 按键1 - 可回收垃圾 | **15** |
| 按键2 - 厨余垃圾 | **4** |
| 按键3 - 有害垃圾 | **18** |
| 按键4 - 其他垃圾 | **19** |

### PCA9685 舵机驱动板 (I2C)

| 垃圾桶 | PCA9685 通道 | 备注 |
|--------|-------------|------|
| 可回收垃圾 | CH15 | 舵机1 |
| 厨余垃圾 | CH11 | 舵机2 |
| 有害垃圾 | CH7 | 舵机3 |
| 其他垃圾 | CH3 | 舵机4 |

### PCF8574 IO 扩展 → 红外反射传感器 (I2C, 地址 0x20)

| 红外传感器 | PCF8574 引脚 |
|-----------|-------------|
| 红外1 (可回收) | P0 |
| 红外2 (厨余) | P1 |
| 红外3 (有害) | P2 |
| 红外4 (其他) | P3 |

> 红外传感器低电平有效 (检测到物体 → 输出低电平)

### HC-SR04 超声波测距传感器

| 传感器 | TRIG | ECHO |
|--------|------|------|
| 超声波1 (可回收) | **13** | **12** |
| 超声波2 (厨余) | **14** | **27** |
| 超声波3 (有害) | **26** | **25** |
| 超声波4 (其他) | **33** | **32** |

> 距离 < 10cm 判定为满桶

### I2C 总线 (默认引脚)

| 信号 | GPIO |
|------|------|
| SDA | **21** |
| SCL | **22** |

I2C 设备列表:
- PCA9685 (舵机驱动板) — 地址 0x40
- PCF8574 (IO 扩展) — 地址 0x20
- SSD1306 (OLED 128×64) — 地址 0x3C

### UART — 天问 ASRPRO 语音模块

| 信号 | ESP32 GPIO |
|------|-----------|
| TX | **17** → ASRPRO RX |
| RX | **16** → ASRPRO TX |
| 波特率 | **115200** |

### 蓝牙

ESP32 内置蓝牙 (Classic BT SPP)，设备名: `ESP32_SmartBin`

## ⚠️ Strapping 引脚提醒

以下引脚是 ESP32 的 Strapping 引脚，启动时有特殊要求：

| GPIO | 功能 | 风险 |
|------|------|------|
| 12 | MTDI — 影响 Flash 电压 (3.3V/1.8V) | 用作超声波 ECHO 输入，确保启动时为低电平或不影响启动 |
| 13 | Strapping 引脚 | 用作超声波 TRIG 输出，启动后正常 |
| 15 | MTDO — 影响启动日志输出 | 用作按键输入 (INPUT_PULLUP)，确保启动时无冲突 |

> 建议: 如果遇到 ESP32 启动异常，请优先检查 GPIO12 和 GPIO15 的外围电路。
