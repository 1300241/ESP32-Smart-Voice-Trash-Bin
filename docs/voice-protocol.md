# 天问 ASRPRO 语音模块通信协议

## 硬件连接

| ESP32 | ASRPRO |
|-------|--------|
| GPIO17 (UART2 TX) | RX (引脚 3) |
| GPIO16 (UART2 RX) | TX (引脚 2) |
| GND | GND |

- 波特率: **115200-8-N-1**
- 电平: 3.3V TTL

## 通信协议

### ESP32 → ASRPRO (发送指令)

格式: `<前缀><指令码>\n`

| 指令 | 含义 |
|------|------|
| `O100\n` | 打开可回收垃圾桶 (播放对应语音) |
| `O101\n` | 打开厨余垃圾桶 |
| `O102\n` | 打开有害垃圾桶 |
| `O103\n` | 打开其他垃圾桶 |
| `C100\n` | 关闭可回收垃圾桶 (播放对应语音) |
| `C101\n` | 关闭厨余垃圾桶 |
| `C102\n` | 关闭有害垃圾桶 |
| `C103\n` | 关闭其他垃圾桶 |

- `O` = Open (打开)
- `C` = Close (关闭)
- 指令码: `100`=可回收, `101`=厨余, `102`=有害, `103`=其他

### ASRPRO → ESP32 (语音识别结果)

格式: `<指令码>\r\n`

| 语音命令 | ASRPRO 返回 | 触发方式 |
|----------|-------------|----------|
| "可回收垃圾" | `100` | 语音识别 |
| "旧书本" | `100` | 语音识别 |
| "厨余垃圾" | `101` | 语音识别 |
| "苹果" | `101` | 语音识别 |
| "有害垃圾" | `102` | 语音识别 |
| "电池" | `102` | 语音识别 |
| "其他垃圾" | `103` | 语音识别 |
| "石头" | `103` | 语音识别 |

### ASRPRO 接收串口指令后的语音播报

| 收到指令 | ASRPRO 播放语音 |
|----------|----------------|
| `O100` | "正在打开可回收垃圾桶，请您投放。" |
| `O101` | "正在打开厨余垃圾桶，请您投放。" |
| `O102` | "正在打开有害垃圾桶，请您投放。" |
| `O103` | "正在打开其他垃圾桶，请您投放。" |
| `C100` | "关闭可回收垃圾桶，请小心。" |
| `C101` | "关闭厨余垃圾桶，请小心。" |
| `C102` | "关闭有害垃圾桶，请小心。" |
| `C103` | "关闭其他垃圾桶，请小心。" |

## 唤醒词

**"聪明蛋"** — 唤醒后 ASRPRO 回复 "我在呢，你是什么垃圾"

## ASRPRO IO 引脚配置

| ASRPRO 引脚 | 功能 | 模式 |
|-------------|------|------|
| IO4 | 状态输出 (打开=HI, 关闭=LO) | 输出 |

> 当前固件中 IO4 用于状态指示，ESP32 侧未读取此信号，可根据需要扩展。

## 天问 ASRPRO 配置参数

- 音色: 小蝶-清新女声
- 音量: 4
- 语速: 10
- 自动休眠: 15 秒无交互后进入休眠
- 唤醒词播报音量: 5
